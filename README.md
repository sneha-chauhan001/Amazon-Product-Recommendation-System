# Amazon Product Recommendation System

## Overview
This project implements and compares three recommendation systems using an Amazon Products Dataset.

The implemented recommendation systems are:
- Content-Based Filtering
- Collaborative Filtering
- Hybrid Filtering

The models are evaluated using Precision, Recall, and F1-score to compare their recommendation performance.

---

## Features

- Data preprocessing and cleaning
- Feature extraction using TF-IDF
- Content-Based Recommendation System
- Collaborative Filtering using User-Item Matrix
- Hybrid Recommendation System
- Performance evaluation using Precision, Recall, and F1-score
- Visualization of comparison results using Matplotlib

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## Algorithms Used

### Content-Based Filtering
- TF-IDF Vectorizer
- Cosine Similarity

### Collaborative Filtering
- User-Item Matrix
- Cosine Similarity

### Hybrid Filtering
- Combination of Content-Based and Collaborative Filtering

---

## Evaluation Metrics

- Precision
- Recall
- F1-score

---

## Results

- Content-Based Filtering achieved the highest performance on this dataset.
- Hybrid Filtering produced similar performance.
- Collaborative Filtering was affected by sparse user-rating data.
